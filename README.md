## node安装
https://nodejs.org/en/download/

node --version
//成功的话输出类似：v10.6.0

npm --version
//成功的话输出类似：6.1.0

npm的下载源切换到国内淘宝的镜像

npm set registry https://registry.npm.taobao.org/

vue
